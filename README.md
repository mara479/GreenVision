# GreenVision
Aplicație Descktop Python (CustomTkinter) pentru ghid de reciclare quiz,statisticv, recenzii si harta(Folium+SQLite)

## Funcționalități
-Ghid de reciclare pe categorii (Plastic, Hârtie, Sticlă, Metal, Baterii)
- Sistem de steluțe și progres, salvat în SQLite
- Quiz educațional cu întrebări random și feedback imediat
- Statistici de performanță (scor maxim, medie, număr quiz-uri)
- Secțiune de recenzii și feedback
- Hartă interactivă a punctelor de colectare (Folium – Timișoara)

## Tehnologii utilizate
- Python  
- CustomTkinter (UI)  
- SQLite (persistență date)  
- Folium (hărți interactive)  
- Pillow (procesare imagini)  
- Matplotlib (statistici – opțional)
